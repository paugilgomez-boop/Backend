# TowerDefence

API REST para gestionar usuarios, items e inventarios de un juego.

## Endpoints principales

- `POST /dsaApp/game/auth/register`
- `POST /dsaApp/game/auth/login`
- `POST /dsaApp/game/items`
- `PUT /dsaApp/game/items/{itemId}`
- `DELETE /dsaApp/game/items/{itemId}`
- `GET /dsaApp/game/items`
- `POST /dsaApp/game/players/{playerId}/inventory`
- `GET /dsaApp/game/players/{playerId}/inventory`
